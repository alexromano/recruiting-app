## Description

This backend uses [Nest](https://github.com/nestjs/nest), a framework for building server-side applications.

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Database
This project uses [MongoDB](https://www.mongodb.com/) as a database. Start the db locally with
```bash
docker-compose up -d
```
